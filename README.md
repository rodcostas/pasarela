## Quick start
1. Upload your images to `/assets/images/`
2. Update `/data/products.json`
3. (Optional) Update `/data/config.json` for contact destination (WhatsApp / Instagram)
4. Push to GitHub and enable Pages.

## Backstage
Open `/admin.html` locally or on GitHub Pages:
- Edit products
- Download `products.json`
- Replace `/data/products.json` in the repo

## Contact
- WhatsApp: set `contact.mode = "whatsapp"` and update `phoneE164`
- Instagram: set `contact.mode = "instagram"` and update `handle`
